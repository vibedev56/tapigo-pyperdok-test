Простое SPA-приложение на Vue 3 для управления списком задач.

## Установка и запуск

  1. Склонируйте репозиторий:

  ```bash 
  git clone https://github.com/vibedev56/tapigo-pyperdok-test.git 
  ```

  2. Установите зависимости:

 ```bash 
  npm install
  ```

  3. Запустите приложение:

  ```bash 
  npm run dev  
  ```
